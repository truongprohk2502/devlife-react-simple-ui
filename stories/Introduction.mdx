# Welcome to DevLife Simple UI

## Introduction

Our UI library is designed to help you build beautiful and consistent user interfaces for your projects. It provides a collection of reusable components and styles that can be easily integrated into any application.

## Getting Started

To get started with using our UI library, follow these steps:

### 1. Install the package using npm or yarn:

```bash
npm install devlife-react-simple-ui
```

### 2. Import the css styles into the root file of your project:

```js
// main.ts
import "devlife-react-simple-ui/dist/style.css";
```

### 3. Start using the components in your application:

```js
import { Button, Input } from "devlife-react-simple-ui";

const App = () => (
  <>
    <Button variant="primary">Click me</Button>
    <Input size="small" placeholder="Enter your email" />
  </>
);
```
